# AI-Powered Resume Keyword Matcher

The AI-Powered Resume Keyword Matcher is a simple tool that helps job seekers check how well their resume matches a job description. It uses OpenAIâ€™s API along with keyword matching and similarity techniques to highlight missing keywords and provide a match score. The app is built with Streamlit, making it easy to use in your browser.

## Features
- Upload resumes in PDF, DOCX, or TXT format
- Automatically extracts text from resumes
- Compare your resume with a job description
- Shows a similarity score and missing keywords
- Easy-to-use interface in your browser
